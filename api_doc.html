GET /user/verify-token 验证令牌有效性

验证用户令牌的有效性，并返回用户角色信息，可用于前端权限控制和验证登录状态。

请求头

参数名 类型 是否必须 描述
Authorization String 必须 Bearer 令牌

响应示例 - 有效令牌

JSON 
{
  "code": 0,
  "data": {
    "valid": true,
    "userId": 3,
    "username": "zhangsan",
    "role": "job_seeker"
  },
  "message": "success"
}

响应示例 - token错误或被篡改

JSON 
{
  "code": 40005,
  "data": null,
  "message": "token 错误或被篡改"
}

响应示例 - token已过期

JSON 
{
  "code": 40006,
  "data": null,
  "message": "token 已失效"
}

说明

  • 接口可区分token是过期还是无效（错误或被篡改），并返回不同的错误码和消息
  • 当用户在应用中操作时，可定期调用此接口检验令牌是否仍然有效
  • 可根据返回的用户角色信息进行前端页面权限控制

企业模块

企业注册、管理和信息检索相关接口

POST /company/register 注册新企业及管理员

注册新企业并创建企业管理员账号。成功返回"注册成功"提示。

请求参数

参数名 类型 是否必须 描述
name String 必须 企业名称（最多100个字符）
licenseNumber String 必须 营业执照号（18位）
address String 可选 企业地址（最多200个字符）
legalPerson String 必须 法定代表人姓名（最多50个字符）
username String 必须 企业管理员用户名（4-50个字符）
password String 必须 企业管理员密码（6-20个字符）
email String 必须 企业管理员电子邮箱地址
position String 可选 企业管理员职位（最多50个字符）

请求示例

JSON 
{
  "name": "北京智慧科技有限公司",
  "licenseNumber": "91110105MA00B4XC2X",
  "address": "北京市海淀区科技园区123号",
  "legalPerson": "李明",
  "username": "admin1",
  "password": "admin123",
  "email": "admin@zhihuikeji.com",
  "position": "总经理"
}

响应示例

JSON 
{
  "code": 0,
  "data": "注册成功",
  "message": "success"
}

项目模块

建筑项目创建、管理和信息检索相关接口

POST /project/add 添加建筑项目

创建新的建筑项目。注意：求职者(job_seeker)角色不允许添加项目。

请求头

参数名 类型 是否必须 描述
Authorization String 必须 Bearer 令牌（项目经理或企业管理员令牌）

请求参数

参数名 类型 是否必须 描述
companyId Integer 必须 关联企业ID
name String 必须 项目名称（最多100个字符）
address String 必须 项目地址（最多255个字符）
startDate Date 必须 开工日期（YYYY-MM-DD格式）
expectedEndDate Date 必须 预计竣工日期（YYYY-MM-DD格式）
projectType String 必须 项目类型（residence：住宅，commerce：商业，industry：工业，municipal：市政）
projectScale String 必须 项目规模（small：小型，medium：中型，big：大型）
totalArea Decimal 必须 总建筑面积（平方米）
budget Decimal 必须 项目预算（元）
description String 可选 项目描述

请求示例

JSON 
{
  "companyId": 1,
  "name": "华安花园住宅小区",
  "address": "北京市朝阳区建国路88号",
  "startDate": "2025-06-01",
  "expectedEndDate": "2026-12-31",
  "projectType": "residence",
  "projectScale": "big",
  "totalArea": 50000.00,
  "budget": 80000000.00,
  "description": "大型住宅小区建设项目，包含10栋高层住宅楼及配套设施"
}

响应示例

JSON 
{
  "code": 0,
  "data": {
    "id": 1,
    "companyId": 1,
    "name": "华安花园住宅小区",
    "address": "北京市朝阳区建国路88号",
    "projectManagerId": 5,
    "projectManagerName": "manager1",
    "startDate": "2025-06-01",
    "expectedEndDate": "2026-12-31", 
    "projectType": "residence",
    "projectScale": "big",
    "totalArea": 50000.00,
    "budget": 80000000.00,
    "description": "大型住宅小区建设项目，包含10栋高层住宅楼及配套设施",
    "status": "pending",
    "createTime": "2025-04-14T10:30:00",
    "updateTime": null
  },
  "message": "success"
}

错误响应示例

JSON 
{
  "code": 40007,
  "data": null,
  "message": "您所属的用户组不允许添加项目"
}
